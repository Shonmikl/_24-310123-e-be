Бизнес-логика: Создать приложение, в котором несколько потоков будут выполнять задачи параллельно.
Главный поток должен ждать, пока все потоки не закончат выполнение своих задач.

Техническое задание:

Создать класс TaskWorker, который будет реализовывать интерфейс Runnable. В этом классе будет задача, которую поток должен выполнить.
Создать класс MainThread, который будет содержать метод main().
В методе main(), создать объект CountDownLatch с количеством задач (потоков), которые будут выполняться.
Создать несколько объектов TaskWorker (потоков) и передать им CountDownLatch для синхронизации.
Запустить все потоки.
Главный поток вызывает метод await() у объекта CountDownLatch для ожидания завершения всех потоков.
